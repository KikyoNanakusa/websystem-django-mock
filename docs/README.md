# Getting Started
1. Docker Desktopを以下からインストール
https://www.docker.com/ja-jp/products/docker-desktop/ 

2. 本リポジトリをクローン
```bash
git clone https://github.com/KikyoNanakusa/websystem-django-mock.git
```

3. プロジェクトのルートに以下の内容で .env  を作成
内容は, 管理者に連絡して確認してください. 

4. 以下のコマンドを実行
```bash
docker compose up --build
```
